# <center>Análise de Estoque</center>

### Objetivo

Este projeto tem como objetivo analisar dados de estoque para identificar padrões, riscos e oportunidades de melhoria na gestão, apoiando a tomada de decisão.

### Ferramentas utilizadas

- Excel
- Power Query (tratamento de dados)
- Power BI (Dashboard interativo)

### Etapas do projeto

- Limpeza e tratamento da base de dados
- Padronização de informações
- Criação de métricas (KPIs)
- Desenvolvimento do dashboard

### Principais Insights

- Alto volume de produtos vencidos: 3,89 Milhões de 7,54 Milhões
    - Foi identificado um volume elevado de itens vencidos, representando uma parcela significativa do estoque.
    - Isso pode indicar falhas no controle de validade e no planejamento de compras, gerando risco de perdas financeiras.

- Concentração de estoque por região
    - O estoque está concentrado em poucos estados, enquanto outras regiões possuem baixa disponibilidade.
    - Essa distribuição pode gerar ineficiências logísticas e impactar o tempo de atendimento.

- Dependência de marca específica
    - Uma das marcas apresenta participação significativamente maior no valor total de estoque: Capa com 1,36 Milhões.
    - Isso pode indicar dependência e aumentar riscos caso haja variações na demanda.

- Baixa representatividade de algumas marcas
    - Algumas marcas possuem participação muito reduzida no estoque, o que pode indicar baixo giro ou ineficiência na alocação de recursos.

### Recomendações
- Revisar o planejamento de compras
- Melhorar o controle de validade dos produtos
- Avaliar a distribuição de estoque entre regiões
- Monitorar o desempenho das marcas com baixa representatividade
